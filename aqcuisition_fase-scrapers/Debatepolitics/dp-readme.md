# DebatePolitics

DP(DebatePolitics) is een politiek forum dat non-biased/on-partijdig is en elk persoon gelijk behandeld bij elk onderwerp.

De DP website heeft de volgende functies;

inloggen/registreren, zoeken(hier kan gefilterd worden op leden van het forum of kan alleen op titels worden gezocht.), blogs(DP maakt zelf ook blogs over verschillende onderwerpen.), search threads(extra filters zoals; keywords, posted_by, newer than, older than, minimal number of replies, prefixes by state, forum selectie, en je kan sorteren op relevantie,datum of meeste reacties).
